Landing page institucional desenvolvida para a empresa Tomazoni Seguros, com foco em apresentar a história, equipe, serviços, diferenciais e canais de contato da corretora.

---

## ✨ Funcionalidades

- **Layout Responsivo:** Visual moderno, adaptado para dispositivos móveis e desktop.
- **Scroll Reveal:** Animações suaves ao rolar a página, destacando conteúdos importantes.
- **Cálculo automático de anos de atuação:** Exibe dinamicamente o tempo de mercado da empresa.
- **Seção de Serviços:** Cards organizados por ramos de atuação, facilitando a navegação.
- **Carrossel de Seguradoras:** Exibição interativa das principais seguradoras parceiras.
- **Links Rápidos para WhatsApp:** Botões para contato direto com setores específicos.
- **Redes Sociais:** Acesso facilitado ao Instagram, LinkedIn e Facebook da empresa.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS** 
- **JavaScript** (puro, sem frameworks)

## 🚀 Como Executar

1. Clone este repositório:
   ```
   git clone https://github.com/DanielBechara/projetoTomazoniSeguros.git
   ```
2. Acesse a pasta do projeto:
   ```
   cd projetoTomazoniSeguros
   ```
3. Abra o arquivo `index.html` em seu navegador preferido.

Não é necessário nenhum servidor ou dependência extra.
